## タイトル  
現在の年齢表示をさせる。

## 概要  
- 西暦の現在から生まれた年を引いて年齢を求める。

## 使用方法  
実行方法の例    
＄./age.py    
[INFO] [1736043329.693131117] [age_calculator_node]: 現在の年齢は 21 歳です。（基準：2004 年生まれ）

## 必要なソフトウェア  
・Python  
・Ubuntu 20.04  

## テスト環境  
・Ubuntu 20.04
